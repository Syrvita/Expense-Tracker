\# Expense Tracker (PHP + MySQL)



\## Features

\- User login/logout

\- Categories CRUD

\- Expenses CRUD + monthly filter

\- Reports (total spent per category) + chart

\- Budget (daily/weekly/monthly) with progress tracking



\## Tech Stack

\- HTML / CSS / JavaScript

\- PHP (MySQLi)

\- MySQL

\- Chart.js



\## Setup (Localhost)

1\. Put the folder in `htdocs`

2\. Start Apache + MySQL in XAMPP

3\. Import SQL file:

&nbsp;  - `database/expense\_tracker.sql`

4\. Edit DB config:

&nbsp;  - `includes/db.php`

5\. Open:

&nbsp;  - `http://localhost/Expense-Tracker/`



\## Folder Structure

See project folders: `/auth /categories /expenses /reports /budget /settings /includes /css /js`
